*****************************Essential Linux Commands*****************************

I have mentioned few essential Linux commands with there syntax and I have also mentioned few options which could be used , I will be updating it on regular basis.

Before proceeding I would also like to tell you that for copying and paste use ctrl+shift+C and ctrl+shift+V respectively.

1. whoami : It prints the current username

2. man : It stands for manual, it displays the manual of the argument passed to it.

Syntax: man [argument]


3. clear : It clears the terminal session and clears scrollback buffer if any.

Options : 
a. -x : prevents from clearing the scrollback buffer, clear -x

4. pwd : It stands for Print Working Directory, displays full file name of current working directory.

5. ls : list directory content. Ignores files & directories starting from a ‘.’ by defualt.

Options: 
a. -l : display long detailed information 
b. -S : sort by file size , largest first 
c. -h : human readable
d. -a : it makes the hidden files display

6. cd : Change Directory

a. To move in parent/previous directory use : cd ..
b. to move in parent’s parent directory use : cd ../..
and so on………..

7. mkdir : make directory(ies) , it creates directory if they do not exist.

           Can pass multiple directories as arguments , like mkdir dir1 dir2 dir3
